# WiiBrowser Lite
A community-driven revival project for "WiiBrowser" by gave92, a homebrew alternative to the Wii's Internet Channel. While not in active development by me, pull requests are welcome.

### Currently supported
- HTML 4.01
- (very basic) CSS
- HTTPS
- Downloads
- Bookmarks
- PNG, JPEG, GIF and BMP
- Web forms
